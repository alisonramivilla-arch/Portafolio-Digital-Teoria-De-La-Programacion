## 💻 Entorno de Desarrollo y Herramientas

He seleccionado las siguientes herramientas para garantizar un flujo de trabajo eficiente y profesional:

| Herramienta | Aplicación |
| :--- | :--- |
| **Visual Studio Code** | Editor de código fuente principal. |
| **GCC (MinGW/Linux)** | Compilación manual mediante comandos de terminal. |
| **Lucidchart** | Diseño de organizadores gráficos y diagramas de flujo. |
